==== mAI friend - Adaptive AI Companion =====

features: 
    - remembers user preferences
    - interactive
    - adapts its tone - Adaptive Personality (Memory)
    - switches personality modes (IMPERSONATION MODE)
    - remembers history
    - trained through prompt engineering

1. Display message -> Personality Selector -> LLM Decide Personality
    -> Prompt Input Depends on Personality -> Prompt Builder -> LLM (Gemini 2.5 flash) 
    -> (Input + Response -> Memory storage) -> Output text response.
2. Mode Select
3. Personality Mode System (personalities.py) -> prompt
4. Prompt Builder (prompt.py) -> personality, history, user_message
5. Memory System (use json and prompt engineering to store memory)
